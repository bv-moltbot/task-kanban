# Task Kanban

A Next.js application with PocketBase integration for task management.

## Tech Stack

- **Frontend**: Next.js 15 with React 19
- **Backend**: PocketBase (BaaS)
- **Styling**: Tailwind CSS
- **Language**: TypeScript

## Getting Started

\`\`\`bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
\`\`\`

## API Health Check

\`GET /api/health\` - Returns server health status

## Deployment

This app is ready for deployment on platforms with Node.js support.

Files at root level for Railpack compatibility:
- start.sh (executable build/start script)
- package.json
- src/
